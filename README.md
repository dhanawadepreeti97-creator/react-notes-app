# 📝 React Notes App

A modern and responsive **Notes Management Application** built using **React.js** and **Vite**. This application allows users to create, edit, delete, pin, and organize notes with a beautiful dark-themed user interface. All notes are automatically saved using **Local Storage**, ensuring data remains available even after refreshing the page.

---

## 🚀 Live Demo

https://react-notes-app-yhj1.vercel.app/

## ✨ Features

- 📝 Create new notes
- ✏️ Edit existing notes
- 🗑️ Delete notes
- 📌 Pin important notes
- 💾 Auto Save using Local Storage
- 📊 Total Notes Counter
- 🎨 Beautiful Dark Theme UI
- 🌈 Colorful Note Cards
- 📅 Date & Time for every note
- 📱 Fully Responsive Design
- ⚡ Built with React + Vite
- 🎯 Clean and Modern Dashboard Layout

## 💻 Installation

### Clone the repository

```bash
git clone https://github.com/dhanawadepreeti97-creator/react-notes-app.git
```

### Go to project folder

```bash
cd react-notes-app
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm run dev
```

The application will start at:

```
http://localhost:3000/
```

## 💾 Local Storage

The application automatically saves all notes in the browser using **Local Storage**, so your notes remain available even after refreshing or reopening the browser.
